The user wants to save something to project memory for future sessions.

1. Read the current DUCK.md (or create one if it doesn't exist)
2. Add a new entry under a "## notes" section at the bottom
3. Format: `- [YYYY-MM-DD] <what to remember>`
4. Keep entries concise (one line each)
5. Do not duplicate existing entries

If the user didn't specify what to remember, ask them.
